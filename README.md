A motus game
============

A simple motus game implemented in Lua.

The motus was a TV game shown on french television. The closest seems to be Lingo in the U.S.

Motus principle :
 - https://fr.wikipedia.org/wiki/Motus_(jeu_télévisé)
 - https://en.wikipedia.org/wiki/Lingo_(U.S._game_show)

This was for an university coursework I've made.
It will not be maintained, it's just for sharing and education purposes.
